# Agent Skills

This directory contains specialized skills that extend the AI's capabilities for coding, communication, and efficiency.

| Skill | Purpose |
|-------|---------|
| caveman | **Terser Communication.** Cuts token usage by ~75% by removing fluff while keeping technical accuracy. |
| caveman-commit | **Smart Commits.** Generates concise, "why-over-what" commit messages using Conventional Commits. |
| caveman-review | **Surgical PR Feedback.** Provides one-line review comments: Location, Problem, and Fix. |
| compress | **Token Optimization.** Compresses text/markdown files into a dense format to save context window space. |
| karpathy-guidelines | **Quality Control.** Behavioral rules to avoid over-engineering and reduce common LLM coding pitfalls. |
